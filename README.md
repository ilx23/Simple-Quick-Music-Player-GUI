## Introduction:

Welcome to the Simple Quick Music Player GUI project! This project provides a straightforward Graphical User Interface (GUI) for playing music files quickly and easily. Users can select their favorite songs and enjoy listening to them without the need for complex controls.

## Features:

1. **User-Friendly Interface:** The GUI offers an intuitive interface for users to select and play music files with minimal effort.

2. **Music Playback:** Users can select music files from their local storage and play them directly within the application.

3. **Play, Pause, and Stop Controls:** The application includes basic playback controls, allowing users to play, pause, and stop the music playback as needed.

## How to Use:

1. **Clone the Repository:** Clone this repository to your local machine using the following command:
> ```git clone https://github.com/ilx23/Simple-Quick-Music-Player-GUI.git```
